## Examen 01


1. Define clase: 
Una clase es un tipo de agrupación de datos definidos por el usuario, actuando como un modelo con la estructura de atributos (variables) y métodos (funciones) que operan sobre esos datos para crear o construir objetos, definiendo sus estados y operaciones o instrucciones que permiten iniciar el objeto.


2. Define encapsulamiento:
Es un principio importante de la Programación Orientada a objetos. Este consiste para controlar la visibilidad y el acceso a datos
internos de una clase, proporcionando una interfaz para su acceso.



3. Define constructor:
Es un método especial ejecutado automáticamente al crear un objeto, garantizando su inicialización y asignando valores iniciales a los atributos, configurando el estado necesario para su uso.



4. Diseña una clase celular con 5 atributos y 3 métodos. Proporciona el .h y el .cpp.
