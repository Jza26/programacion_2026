# Examen 02

1. Explica polimorfismo.
La palabra polimorfismo significa tener muchas formas. Es la capacidad de objetos de diferentes clases derivadas de tratar comportamientos a través de una interfaz común, generalmente una clase base. Permite que el mismo método responda de formas distintas según el tipo de objeto real.
